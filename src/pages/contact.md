# Contact Us

We welcome questions, feedback, and contributions from the community. Here's how to get in touch:

## For Content Issues & Contributions

We **strongly encourage** using GitHub for all content-related matters:

### Report Issues or Errors
Found a mistake in our documentation? Outdated information? Broken links?

**[Open an issue on GitHub](https://github.com/secretlessorg/secretless.cloud/issues/new)**

This helps us track problems and allows the community to see what's being addressed.

### Suggest Improvements or New Content
Have ideas for new guides? Want to request coverage of a specific provider or integration?

**[Create a feature request](https://github.com/secretlessorg/secretless.cloud/issues/new)** on GitHub.

### Contribute Directly
Want to add content, fix errors, or improve existing documentation?

**[Submit a pull request](https://github.com/secretlessorg/secretless.cloud/pulls)** on GitHub.

We review all contributions and work with you to ensure quality and accuracy.

## What to Expect

- **GitHub Issues/PRs**: We typically respond within 2-3 business days
- **Email**: Please allow 3-5 business days for a response
- **Community-Driven**: This is an open-source project maintained by volunteers, so response times may vary
